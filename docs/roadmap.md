# Roadmap

## v0.1.0 — Linux manager MVP

Configuration, pinned gateway install, device OAuth, session mode, Linux systemd,
stable launcher, diagnostics, permissions, and rollback.

## v0.2.0 — Claude Code plugin UX

Guided setup/login/config/status/doctor/update/uninstall and relaunch workflows.

## v0.3.0 — Compatibility and release hardening

Presets, output/context policy, compatibility promotion, protected live smoke
workflow, release bundles, checksums, and SBOM.

## v0.4.0 — macOS

Intel and Apple Silicon artifacts, session mode, LaunchAgent, and platform OAuth
validation.

## v0.5.0 — Windows

Stable PowerShell launcher, session lifecycle, diagnostics, update/rollback, and
uninstall.

## Future evaluation

A minimal gateway fork or replacement is considered only after measured evidence
shows that CLIProxyAPI's broader surface is the dominant maintenance or security
cost. It is not on the MVP critical path.
